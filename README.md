# Weather Forecast Web App

![Project Banner](https://placehold.co/1200x400/1E88E5/FFFFFF/png?text=Weather+Forecast+App\nReact.js)

A responsive and dynamic web application built with **React.js** that provides real-time weather data for any city in the world. This app features a clean user interface, dynamic updates, and a search history feature for an enhanced user experience.

---

## 📜 Description

This project leverages the [OpenWeatherMap API](https://openweathermap.org/api) to fetch and display current weather conditions, forecasts, and other meteorological details. It was built to demonstrate proficiency in front-end development, specifically in making API calls, managing application state, and creating a responsive user interface that works seamlessly across different devices.

---

## ✨ Features

* **Real-Time Weather Data**: Get up-to-date weather information for any city.
* **City Search**: Easily search for cities worldwide to get their weather forecasts.
* **Dynamic UI Updates**: The interface changes dynamically to reflect the current weather (e.g., different icons and backgrounds for sunny, rainy, cloudy conditions).
* **Search History**: Stores your recent searches in local storage for quick access.
* **Responsive Design**: A mobile-first design that looks great on desktops, tablets, and smartphones.
* **Detailed Information**: Displays temperature, humidity, wind speed, and a 5-day forecast.

---

## 🛠️ Tech Stack

* **Frontend**: React.js, HTML5, CSS3
* **API**: OpenWeatherMap API
* **State Management**: React Hooks (`useState`, `useEffect`)

## 🚀 Getting Started

Follow these instructions to get the project running locally.

### Prerequisites

* Node.js and npm (or yarn)
* An API key from [OpenWeatherMap](https://openweathermap.org/appid)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/React-Weather-Forecast-App.git](https://github.com/your-username/React-Weather-Forecast-App.git)
    cd React-Weather-Forecast-App
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or if you use yarn
    # yarn install
    ```

3.  **Set up environment variables:**
    * Create a file named `.env.local` in the project's root directory.
    * Add your OpenWeatherMap API key to this file:
        ```
        REACT_APP_OPENWEATHERMAP_API_KEY="your_api_key_here"
        ```

4.  **Run the Application:**
    * Start the development server:
        ```bash
        npm start
        # or
        # yarn start
        ```
    * Open your browser and navigate to `http://localhost:3000` to see the app running.
